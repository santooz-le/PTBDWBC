Dropdown com CSS

Esta página demonstra um menu dropdown feito apenas com CSS usando :hover e :focus-within (acessível via teclado).

Como funciona

  • O submenu fica escondido com display: none.
  • Ao passar o mouse no item (.dropdown:hover) o submenu aparece.
  • Para teclado, usamos .dropdown:focus-within para abrir quando um link dentro do dropdown recebe foco.

Trecho de CSS (já aplicado em css/menu.css)

.dropdown__menu { display: none; }
.dropdown:hover .dropdown__menu,
.dropdown:focus-within .dropdown__menu {
  display: block;
}
